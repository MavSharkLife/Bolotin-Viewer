# Bolotin Viewer

Bolotin Viewer is a Flask based application that assists with those with vision impairments.


## Demo
You can view a demo of the project here: https://see.liquidpcs.com
## Installation


```bash
git clone https://github.com/mavsharklife/bolotin-viewer.git
cd bolotin-viewer
pip install -r requirements.txt
cd bolotinviewer
python main.py
```
> The webapp will be running on localhost:5000

### Dependencies
| Package | Version |
|---|---|
| Flask | 3.1.3 |
| numpy | 2.4.2 |
| opencv-python | 4.13.0.92 |
| torch | 2.10.0 |
| ultralytics | 8.4.14 |
